# Pick & Go — SEO Action Plan (to rank #1 for your brand name)

**Created:** 2026-06-24

## The situation (read this first)

Your website is **not in Google's index yet**. That is the real reason nothing shows up when you
search "pick and go rentals" — not bad website SEO. A `site:pickandgorentals.com` search returns
zero pages. The site itself is healthy and fully crawlable.

Two things get you to #1 for your **brand name** (a low-competition search, very winnable):
1. Get Google to **index** the site.
2. Build **off-site presence** — above all, a **Google Business Profile**.

Realistic timeline: once both are done, your brand search should show you within **days to a few
weeks**. (Ranking for generic terms like "car rental miami" is a separate, long-term effort — not
the goal here.)

The website code changes are already done and deployed. The steps below are the ones only **you**
can do, in priority order.

## 🔴 STEP 1 — Get indexed (do today, ~10 min)

1. Go to **search.google.com/search-console** → select the `pickandgorentals.com` property.
2. Left sidebar → **Sitemaps** → if `sitemap.xml` isn't listed (or shows old date), type
   `sitemap.xml` and click **Submit**.
3. Top search bar → paste `https://www.pickandgorentals.com/` → press Enter → click
   **Request Indexing**.
4. Repeat step 3 for `https://www.pickandgorentals.com/es/`.

This is the single most important step. Without it, nothing else matters.
